> **<mark style="background: #0969DA; color: white;">Фишки</mark>** — это неочевидные функции игры, знания о которых труднодоступны, к ним сложно прийти интутивно или которые можно легко упустить в перегруженных возможностями настройках.

## 1. Горячие клавиши

Среди огромного списка горячих клавиш, стоит упомянуть самые полезные из них:

| Сочетание клавиш                 | Действие                                                     |
| -------------------------------- | ------------------------------------------------------------ |
| `F12`                            | Сделать скриншот                                             |
| `Ctrl+T`                         | Скрыть панель инструментов (строка сверху)                   |
| `Ctrl+P`                         | Открыть свой профиль                                         |
| `Ctrl+O`                         | Открыть настройки                                            |
| `Ctrl+B`                         | Открыть библиотеку карт                                      |
| `Shift+TAB`                      | Скрыть интерфейс во время игры                               |
| `Ctrl+Shift+T`<br>`Ctrl+Shift+E` | Переключаться между скинами по порядку вперед/назад          |
| `Ctrl+Enter`                     | Посмотреть идеальное прохождение карты                       |
| `Ctrl+↑`<br>`Ctrl+↓`             | Изменить скорость карты (автоматически применить моды NC/DT) |
| `Ctrl+F4`                        | Выключить звук игры                                          |

## 2. Изменение громкости

Изменять громкость можно:

- Через наведение курсором на иконку музыкальной ноты (плеера) и прокрутом колесика 
- Прокручиванием колесика с зажатой клавишей `Alt`
- Предыдущий способ, затем отпустить `Alt`, навестись курсором на “эффекты/музыка” и прокрутить колесико

## 3. Посмотреть фон карты

Увидеть фон карты в меню выбора карты без элементов интерфейса можно зажатием `ЛКМ` вне элементов интерфейса

## 4. Абсолютная прокрутка списка карт

Быстро прокручивать карты в меню выбора карт можно с зажатой клавишей `ПКМ` вне элементов интерфейса

## 5. Вернуться к карте после прокрутки

Выбрал карту, затем продолжил скроллить в поисках какой-нибудь другой, но все же передумал и захотел сыграть ранее выбранную карту? Чтобы снова к ней вернуться, можно навестись курсором на панель с информацией о карте (слева сверху)

## 6. Быстрое удаление карт

Ускорить удаление сразу нескольких карт можно, если в диалоговом окне вместо удержания кнопки удаления нажимать `ПРОБЕЛ`

## 7. Быстрая загрузка карт

Скачивать карты в библиотеке карт можно не только через кнопку, но и через `Shift+ЛКМ` прямо по карте

## 8. Добавить на экран свою картинку

Чтобы поместить на экран меню выбора карты или экран геймплея свою картинку, можно просто перетащить ее из системы в окно редактора скина `Ctrl+Shift+S` 

## 9. Файлы карты/скина

Посмотреть все файлы внутри карты/скина можно через строку меню → Файл → Изменить в папке. В системе откроется папка, в которой будут доступны все ее элементы для копирования, изменения или удаления

## 10. Продвинутый поиск карт

В меню выбора карты можно фильтровать карты по огромному числу параметров:

- [По различным характеристикам карты](https://osu.ppy.sh/wiki/ru/Beatmap_search#%D0%BA%D0%BB%D0%B8%D0%B5%D0%BD%D1%82-(lazer))
- [По пользовательским тегам](https://osu.ppy.sh/wiki/ru/Beatmap/Beatmap_tags#%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5-%D1%82%D0%B5%D0%B3%D0%B8)

## 11. Список доступных команд в чатах

Узнать о всех доступных командах можно, введя в чат команду /help. Краткое пояснение, что делают эти команды:

| Команда       | Действие                                                                                     |
| ------------- | -------------------------------------------------------------------------------------------- |
| /me [текст]   | Отправить в чат наклонный текст с тем, что ты сейчас “делаешь”                               |
| /join [канал] | Присоединиться к каналу в чате                                                               |
| /chat [игрок] | Открыть личные сообщения с игроком                                                           |
| /np           | Отправить в чат ссылку на карту, которая сейчас играет                                       |
| /savelog      | Выгрузить всю историю чата в текстовый файл                                                  |
| /roll [2-100] | Выбирает случайное число в указанном диапазоне, начиная с 1 (только в комнатах мультиплеера) |

## 12. Тестировщик задержки

В самом конце настроек игры спрятана мини-игра “тестировщик задержки”, которая помогает разубедить игроков, считающих, что ограничение FPS влияет на задержку между устройствами ввода и игрой. На каждом уровне предлагается выбрать ту половину экрана, которая ощущается лучше, до момента, пока вы не начнете ошибаться. После первой ошибки игра предложит пройти сертификацию за 20 раундов, чтобы определить оптимальное ограничение кадров на твоем ПК (игра установит его автоматически).
Команда разработчиков не рекомендует играть на опции “Basically unlimited” (1000 fps) и объясняет почему [в своей статье](https://vk.ru/@osu.content-lazerlatencyfps)

## 13. Кастомные режимы игры

Помимо стандартных 4 режимов игры, есть и другие игры, основанные на фреймворке osu!. Фреймворк (англ. framework — «каркас, структура») — это готовый набор инструментов, который облегчает разработку программного обеспечения. Таких режимов насчитывается минимум 16 штук, узнать о них можно на сайте [rūrusetto](https://rulesets.info/status), а общая тема обсуждения для разработчиков есть на гитхабе [Custom Rulesets Directory #13096](https://github.com/ppy/osu/discussions/13096)

## 14. ВКонтакте внутри игры

В начале 2022 года русский разработчик Feodor0090 разработал “режим игры” ovk tab, который позволял войти во вконтакте прямо внутри игры. Пост об этом спустя год появился в osu!content: [ссылка на пост](https://vk.ru/wall-101391911_114114). В свежих версиях игры этот “режим игры”, увы, не работает. Протестировать можно на гитхабе: [ovktab](https://github.com/Feodor0090/ovktab)

## 15. Пасхалка “Toggles” в настройках

Каждый год, 1 апреля, в настройках появляется сатирическая секция Toggles, призванная  и посмеяться над абсурдной гиперболой в ответ на непрекращающийся поток просьб за все годы разработки лазера о возможности включения/отключения чего-нибудь по всей игре с помощью переключателя в настройках и тем самым “удовлетворить” желание сообщества полностью кастомизировать игру. Секция впервые появилась 1 апреля 2026 года. Вызвать эту секцию можно в любой момент, поменяв в системе дату на 1 апреля. Посмотреть, как это реализовано, можно в группе вк osu!content: [ссылка на пост](https://vk.ru/wall-101391911_150649)

## 16. Инструменты разработчика

Прямо в игре доступны некоторые инструменты разработчика:

| Сочетание клавиш | Инструмент                                        |
| ---------------- | ------------------------------------------------- |
| `Ctrl+F1`        | Визуализатор отрисовки (Draw Visualiser)          |
| `Ctrl+F2`        | Сводка технической статистики (Global Statistics) |
| `Ctrl+F3`        | Список используемых текстур (Textures)            |
| `Ctrl+F9`        | Аудио микшер (Audio Mixer)                        |
| `Ctrl+F10`       | Отладочные логи (Debug Logs)                      |
| `Ctrl+F11`       | Подробная статистика по кадрам                    |

## 17. Скрыть бегущую строку (следующий релиз лазера после 2026.401.0)

Скрыть бегущую строку можно только вместе с настройками записей игр сочетанием клавиш `Ctrl+H`
