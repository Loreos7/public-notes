> **<mark style="background: #0969DA; color: white;">Фишки</mark>** — это неочевидные функции игры, знания о которых труднодоступны, к ним сложно прийти интутивно или которые можно легко упустить в перегруженных возможностями настройках, а также те, которых может не быть в других играх.

## 1. Горячие клавиши

Среди огромного списка горячих клавиш, стоит упомянуть самые полезные из них:

| Сочетание клавиш                 | Действие                                                     |
| -------------------------------- | ------------------------------------------------------------ |
| `F12`                            | Сделать скриншот                                             |
| `Ctrl+T`                         | Скрыть панель инструментов (строка сверху)                   |
| `Ctrl+P`                         | Открыть свой профиль                                         |
| `Ctrl+O`                         | Открыть настройки                                            |
| `Ctrl+B`                         | Открыть библиотеку карт                                      |
| `Shift+TAB`                      | Скрыть интерфейс во время игры                               |
| `Ctrl+H`                         | Скрыть настройки записи игры и бегущую строку                |
| `Ctrl+Shift+T`<br>`Ctrl+Shift+E` | Переключаться между скинами по порядку вперед/назад          |
| `Ctrl+Enter`                     | Посмотреть идеальное прохождение карты                       |
| `Ctrl+↑`<br>`Ctrl+↓`             | Изменить скорость карты (автоматически применяет моды NC/DT) |
| `Ctrl+F4`                        | Выключить звук игры                                          |

## 2. Изменение громкости

Изменять громкость можно:

1. Через наведение курсором на иконку музыкальной ноты (плеера) и прокруткой колесика 
2. Прокруткой колесика с зажатой клавишей `Alt`
3. Предыдущий способ, затем отпустить `Alt`, навестись курсором на “эффекты/музыка” и прокрутить колесико
4. Комбинацией клавиш `Alt+↑` и `Alt+↓`, а переключаться на эффекты и музыку через `Alt+←` и `Alt+→`

## 3. Посмотреть фон карты

Увидеть фон карты в меню выбора карты без элементов интерфейса можно зажатием `ЛКМ` вне элементов интерфейса

## 4. Абсолютная прокрутка списка карт

Быстро прокручивать карты в меню выбора карт можно, двигая мышкой вверх-вниз с зажатым `ПКМ` вне элементов интерфейса. Моментально перейти к любой части списка также можно, нажав `ПКМ` в месте, которое визуально соотносится со списком, если бы он был сжат до размера твоего экрана 

## 5. Вернуться к карте после прокрутки

Выбрал карту, затем продолжил скроллить в поисках какой-нибудь другой, но все же передумал и захотел сыграть ранее выбранную карту? Чтобы снова к ней вернуться, наведись курсором на панель с информацией о карте (слева сверху)

## 6. Быстрое удаление карт

Чтобы удалить несколько карт подряд, нужно последовательно нажать `F3` → `2` → `1`. Вместо `1` можно также нажимать `ПРОБЕЛ`

## 7. Быстрая загрузка карт

Скачивать карты в библиотеке карт можно не только через кнопку, но и через `Shift+ЛКМ` прямо по карте

## 8. Восстановление случайно удаленной карты

Случайно удалил карту? Пока ты не вышел из игры, ее можно восстановить в настройках  `Управление` → `Восстановить недавно удаленные карты`

## 9. Предыдущая случайная карта

Случайная карта `F2` пользуется огромной популярностью, однако не все знают, что вернуться к предыдущей карте из всего списка выбранных случайных карт можно:

1. Комбинацей клавиш `Shift+F2`
2. Нажатием `ПКМ` по кнопке `Случайная` на нижней панели
3. Нажатием `ЛКМ` по кнопке `Предыдущая` с зажатым `Shift`

## 10. Добавить на экран свою картинку

Чтобы поместить на экран меню выбора карты или экран геймплея свою картинку, можно просто перетащить ее из системы прямо в окно редактора скина `Ctrl+Shift+S` 

## 11. Импорт карт, скинов, рекордов

Импортировать скачанные из интернета карты, скины и рекорды можно не только открытием файлов через `ЛКМ`, но и перетаскиванием их прямо в окно игры

## 12. Экспорт рекорда

Хочешь показать другу рекорд, который ты поставил на карте? Сделать это можно двумя способами:

1. Выбери локальную таблицу рекордов на карте `Рейтинг` → `Локальный`, затем кликни по рекорду `ПКМ` → `Экспортировать`
2. Кликни на рекорд и нажми `F2`

## 13. Режим наблюдения

Смотреть, как играют топовые игроки/друзья в прямом эфире можно через `F9` (главная страница) → `друзья`/`сейчас онлайн` → `ПКМ` → `Наблюдать`

## 14. Файлы карты/скина

Посмотреть все файлы внутри карты/скина можно через строку меню → `Файл` → `Изменить в папке`. В системе откроется папка, где будут доступны все ее элементы для копирования, изменения или удаления

## 15. Продвинутый поиск карт

В меню выбора карты можно фильтровать карты по огромному числу параметров:

- [По различным характеристикам карты](https://osu.ppy.sh/wiki/ru/Beatmap_search#%D0%BA%D0%BB%D0%B8%D0%B5%D0%BD%D1%82-(lazer))
- [По пользовательским тегам](https://osu.ppy.sh/wiki/ru/Beatmap/Beatmap_tags#%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5-%D1%82%D0%B5%D0%B3%D0%B8)

## 16. Команды в чатах

Узнать о всех доступных командах можно, введя в чат команду /help. Краткое пояснение, что делают эти команды:

| Команда       | Действие                                                                                     |
| ------------- | -------------------------------------------------------------------------------------------- |
| /me [текст]   | Отправить в чат наклонный текст с тем, что ты сейчас “делаешь”                               |
| /join [канал] | Присоединиться к каналу в чате                                                               |
| /chat [игрок] | Открыть личные сообщения с игроком                                                           |
| /np           | Отправить в чат ссылку на карту, которая сейчас играет                                       |
| /savelog      | Выгрузить всю историю чата в текстовый файл                                                  |
| /roll [2-100] | Выбирает случайное число в указанном диапазоне, начиная с 1 (только в комнатах мультиплеера) |

## 17. Быстродействие

Играешь несколько часов и вдруг заметил, что игра стала работать хуже? Игру можно не перезапускать, вместо этого достаточно очистить кеш в настройках `Debug` → `Clear all caches`

## 18. Тестировщик задержки

В самом конце настроек игры спрятана мини-игра “тестировщик задержки”, которая помогает разубедить игроков, считающих, что ограничение FPS влияет на задержку между устройствами ввода и игрой. На каждом уровне предлагается выбрать ту половину экрана, которая ощущается лучше, до момента, пока вы не начнете ошибаться. После первой ошибки игра предложит пройти сертификацию за 20 раундов, чтобы определить оптимальное ограничение кадров на твоем ПК (игра установит его автоматически).
Команда разработчиков не рекомендует играть на опции “Basically unlimited” (1000 fps) и объясняет почему [в своей статье](https://vk.ru/@osu.content-lazerlatencyfps)

## 19. Кастомные режимы игры

Помимо стандартных 4 режимов игры, есть и другие игры, основанные на фреймворке osu!. 

> <mark style="background: #0969DA;">Фреймворк</mark> (англ. framework — «каркас, структура») — это готовый набор инструментов, который облегчает разработку программного обеспечения.

Таких режимов насчитывается минимум 16 штук, узнать о них можно на сайте [rūrusetto](https://rulesets.info/status), а общая тема обсуждения для разработчиков есть на гитхабе: [Custom Rulesets Directory #13096](https://github.com/ppy/osu/discussions/13096)

## 20. ВКонтакте внутри игры

В начале 2022 года русский разработчик Feodor0090 разработал “режим игры” ovk tab, который позволял войти во вконтакте прямо внутри игры. Пост об этом спустя год появился в osu!content: [ссылка на пост](https://vk.ru/wall-101391911_114114). В свежих версиях игры этот “режим игры”, увы, не работает. Протестировать можно на гитхабе: [ovktab](https://github.com/Feodor0090/ovktab)

## 21. Пасхалка “Toggles” в настройках

Каждый год, 1 апреля, в настройках появляется сатирическая секция `Toggles`, призванная  и посмеяться над абсурдной гиперболой в ответ на непрекращающийся поток просьб за все годы разработки лазера о возможности включения/отключения чего-нибудь по всей игре с помощью переключателя в настройках и тем самым “удовлетворить” желание сообщества полностью кастомизировать игру. Секция впервые появилась 1 апреля 2026 года. Вызвать эту секцию можно в любой момент, поменяв в системе дату на 1 апреля. Посмотреть, как это реализовано, можно в группе вк osu!content: [ссылка на пост](https://vk.ru/wall-101391911_150649)

## 22. Инструменты разработчика

Прямо в игре доступны некоторые инструменты разработчика:

| Сочетание клавиш | Инструмент                                        |
| ---------------- | ------------------------------------------------- |
| `Ctrl+F1`        | Визуализатор отрисовки (Draw Visualiser)          |
| `Ctrl+F2`        | Сводка технической статистики (Global Statistics) |
| `Ctrl+F3`        | Список используемых текстур (Textures)            |
| `Ctrl+F9`        | Аудио микшер (Audio Mixer)                        |
| `Ctrl+F10`       | Отладочные логи (Debug Logs)                      |
| `Ctrl+F11`       | Подробная статистика по кадрам                    |
